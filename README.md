# YouTube → Spotify 변환 크롬 확장

노래 여러 곡이 담긴 유튜브 "노래 모음"(컴필레이션) 영상 하나에서 트랙리스트를 읽어,
각 곡을 Spotify에서 찾아 **영상 제목과 같은 이름의 Spotify 재생목록**을 자동으로
만들어 주는 Manifest V3 크롬 확장 프로젝트.

## 현재 상태
기획/스코프 확정 단계 (개발 착수 전)

## MVP 스코프
- **트랙리스트가 있는 영상만** 지원
  - 소스 1: 유튜브 자동 "음악" 섹션 (원제목·아티스트, Content ID 기반)
  - 소스 2: 설명란 / 고정 댓글 / 챕터의 타임스탬프 트랙리스트
- 트랙리스트 없는 영상 → "지원 안 됨" 처리 (오디오 인식은 향후 단계)

## 매칭 전략
- 소스 우선순위: **자동 "음악" 섹션(원제목) > 설명란(번역·보조)**, 타임스탬프로 병합
- 정제 → Spotify 검색 → 아티스트·제목·재생시간 비교로 신뢰도 점수 → 베스트 매치
- 낮은 신뢰도/미검출 → "확인 필요" 목록으로 분리 (우타이테/커버는 Spotify에 없을 수 있음)

## 기술 스택 (예정)
- Manifest V3 (팝업 UI + 백그라운드 서비스 워커)
- Spotify Web API — OAuth Authorization Code + PKCE, `playlist-modify-public/private`
- 유튜브 — 페이지 파싱 또는 YouTube Data API v3

## 로드맵
1. Spotify OAuth + 재생목록 생성 (쓰기 경로 증명)
2. 유튜브 영상 → 트랙리스트 추출
3. 매칭 엔진 (정제 + 검색 + 신뢰도)
4. 병합 + 배치 추가 + 재생목록 네이밍
5. UI (진행률·미매칭 검토·재시도)
6. 캐싱·에러/레이트리밋 처리
