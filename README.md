# MIXTAPE — YouTube → Spotify 변환기

유튜브 **'노래 모음 / 플레이리스트'** 영상 하나에서 안에 담긴 곡들을 알아내,
각 곡을 Spotify에서 찾아 **영상 제목과 같은 이름의 Spotify 재생목록**을 자동으로
만들어 주는 크롬 확장(Manifest V3)입니다.

> 예: "별이 쏟아진 날 · 잔잔한 우타이테 모음" 영상 → 같은 이름의 비공개 Spotify 재생목록

현재 **v0.8.0 (테스트판)** — 지인 대상 비공개 테스트 중입니다.

---

## ✨ 주요 기능

- **여러 소스에서 트랙리스트 자동 인식** — 설명란 타임스탬프, 유튜브 챕터, 고정 댓글,
  유튜브 자동 감지 "음악" 섹션 중 하나만 있으면 변환됩니다.
- **똑똑한 매칭** — 아티스트 엔티티(ID) 확정으로 표기 차이(로마자↔일본어 등)를 흡수하고,
  제목·아티스트·**재생 시간**을 종합해 신뢰도를 매깁니다. 커버/리믹스/노래방 버전은 감점.
- **원곡·최초 릴리스 우선** — 같은 곡의 여러 발매본 중 컴필·리마스터를 걸러 최초 발매를 선택.
- **검토 UI** — 애매한 곡은 후보를 미리듣기(Spotify 임베드)하고 직접 고르거나 건너뛸 수 있고,
  직접 검색도 됩니다. 원본과 길이가 같은 후보엔 **'길이 일치'** 배지가 붙습니다.
- **원래 순서 보존** — 검토 곡을 나중에 추가해도 유튜브 트랙리스트 순서 그대로 재생목록에 쌓입니다.
- **지연 생성** — 검토를 마치고 **[재생목록 만들기]** 를 누르는 시점에 한 번에 생성해,
  중간에 그만둬도 반쪽짜리 재생목록이 남지 않습니다.

## 🧩 동작 방식 (요약)

1. 콘텐츠 스크립트가 현재 유튜브 영상에서 트랙리스트를 추출합니다
   (설명란 → 챕터 → 고정/상단 댓글 → 자동 "음악" 카드 순 폴백).
2. 서비스 워커가 각 곡을 Spotify Web API로 검색·채점해 **자동 추가 / 확인 필요 / 못 찾음** 3계층으로 분류합니다.
3. 사용자가 '확인 필요'를 검토한 뒤, 그 시점의 선택 전체를 원래 순서대로 재생목록으로 생성합니다.

## 🔐 권한 · 프라이버시

- 인증: Spotify **OAuth Authorization Code + PKCE** (클라이언트 시크릿 없음).
  스코프는 재생목록 생성/수정과 국가 확인(`user-read-private`)에 한정됩니다.
- 확장은 **youtube.com**(트랙리스트 읽기)과 **api/accounts.spotify.com**(매칭·생성)에만 접근합니다.
- 토큰은 브라우저 로컬 저장소(`chrome.storage.local`)에만 보관되며 외부로 전송되지 않습니다.
- 생성되는 재생목록은 기본 **비공개**입니다.

## 🛠 기술 스택

- Manifest V3 — 팝업 + 서비스 워커 + 전용 탭(진행·검토·생성)
- Spotify Web API (2026-02 마이그레이션 반영: `/me/playlists`, `/playlists/{id}/items`)
- 순수 매칭 로직은 의존성 없는 바닐라 JS (`src/lib/matching.js`, `parse-tracklist.js`) — Node로 테스트

## 🧪 테스트

의존성 없이 Node만으로 실행됩니다. `chrome`·`fetch`·DOM을 모두 목(mock)으로 대체해
실제 소스를 로드·검증하므로 Spotify API 호출은 0회입니다.

```bash
for f in test/*.test.js; do node "$f"; done
```

## 📦 설치 (테스트판)

크롬 웹스토어 등록 전이라 **압축 해제 후 직접 로드**하는 방식입니다.

> **먼저!** Spotify 연결은 개발자 대시보드에 **등록된 계정만** 가능합니다(테스트 단계, 최대 25명).
> 사용할 Spotify 계정 이메일을 제게 먼저 알려주세요. 등록은 1분이면 됩니다.

1. [Releases](../../releases)에서 최신 zip을 받아 **압축을 해제**합니다.
   (폴더를 지우면 확장도 꺼지니 안 지울 위치에 두세요.)
2. 크롬 주소창에 `chrome://extensions` 입력
3. 오른쪽 위 **개발자 모드** 켜기
4. **압축해제된 확장 프로그램을 로드** → 압축 푼 폴더 선택
5. 툴바에 초록색 아이콘이 보이면 완료
   (안 보이면 퍼즐 조각 아이콘 → 고정핀 클릭)

## ▶️ 사용법

1. 유튜브에서 노래 모음 / 플레이리스트 영상을 엽니다.
2. 확장 아이콘 클릭 → **[Spotify 연결]** (최초 1회)
3. **[재생목록으로 변환]** → 자동 매칭 진행
4. '확인 필요' 곡은 후보를 눌러 미리듣고 추가 / 건너뛰기 (직접 검색도 가능)
5. **[재생목록 만들기]** → 영상 제목과 같은 이름의 비공개 재생목록 생성

**인식되는 영상**: 설명란·챕터·고정 댓글에 트랙리스트가 있거나, 유튜브가 곡을 자동 인식한 영상.
넷 다 없으면 "지원 안 됨"이 정상입니다.

**팁**: 영상을 클릭으로 이동한 직후 인식이 이상하면 새로고침(F5) 한 번. 변환 탭을 닫아도
백그라운드에서 계속됩니다. 뭔가 꼬였으면 변환 화면 오른쪽 위 **[초기화]** 로 처음부터 다시.
