# YouTube → Spotify 변환기 · DESIGN.md

이 확장의 디자인 시스템 명세. 모든 UI는 이 문서를 grounding으로 만든다.
(Claude Design 방식: 토큰·규칙·근거를 한 파일에. AI 양산형 클리셰는 §7에서 명시적으로 금지.)

## 1. 비주얼 테마 & 분위기
Spotify에 네이티브하게 녹아드는 느낌. 조용하고, 자신감 있고, **콘텐츠 우선**.
UI 자체는 뒤로 물러나고 — **앨범아트와 곡 제목이 유일한 색**이다. 어둡고, 플랫하고, 고대비.
장식을 위한 장식 없음.

## 2. 색 팔레트 & 역할 (전부 솔리드 — 그라데이션 없음)
```
--bg:        #000000   /* 페이지 바탕 */
--surface:   #121212   /* 앱 표면 */
--surface-2: #181818   /* 카드 */
--surface-3: #242424   /* 행 hover / 입력 */
--green:     #1ed760   /* 오직 주요 액션에만 */
--green-press:#19b84c
--text:      #ffffff
--text-2:    #b3b3b3
--text-3:    #6a6a6a
--review:    #e8a33d   /* 확인필요 — 플랫 텍스트/뱃지만, 글로우 금지 */
--miss:      #e4586a   /* 못찾음 */
--line:      #2a2a2a   /* 헤어라인 구분선 */
```
초록은 **주요 액션 하나에만**. 상태색(주황/빨강)은 작은 텍스트·뱃지로만. 나머지 색은 앨범아트에서 온다.

## 3. 타이포그래피
- Family: `Pretendard`(한/영), `Noto Sans JP` 폴백. **Inter/DM Sans/시스템 기본 금지.**
- 스케일: display 22·800 / title 15·700 / body 13·500 / label 11·700(대문자, tracking .09em) / caption 11·500
- 제목은 굵고 tracking을 살짝 좁게(-.01em). 보조 정보는 `--text-2`.

## 4. 컴포넌트
- **버튼**: pill(radius 999px). Primary = 솔리드 `--green` + 검정 글자 800. Secondary = 투명 + 1px `#535353`. 그라데이션/글로우 금지. hover는 밝기 변화만(bounce 금지).
- **카드**: 솔리드 `--surface-2`, radius 8px, 1px `--line` 또는 무테. **그림자로 글로우 내지 말 것.** 계층은 표면 색으로.
- **행(row)**: 플랫, 사이는 헤어라인 구분선, hover = `--surface-3`.
- **입력**: 솔리드 `--surface-3`, 1px `#333`, radius 6~999px.
- **앨범아트**: 선명한 정사각, radius 4~6px, 광택 오버레이 금지.

## 5. 레이아웃
- 간격 스케일: 4 / 8 / 12 / 16 / 24 / 32.
- 팝업 380px 고정. 전용 탭은 최대 560px 중앙 정렬.
- 섹션은 대문자 라벨로 구분. 콘텐츠 우선 밀도(과한 여백 금지).

## 6. 깊이 & 입체
**플랫이 원칙.** 계층 구분은 **솔리드 표면 색**(#121212 < #181818 < #242424)으로. 그림자/블러/글로우 아님.
떠 있는 요소(주요 CTA 등)에만 아주 옅은 그림자 1개 허용. inner glow·그라데이션 보더 금지.

## 7. DO / DON'T (AI 양산형 방지 가드레일)
**DO** — 솔리드 플랫 표면 · 초록은 주요 액션에만 · 앨범아트를 유일한 색으로 · 고대비 · 헤어라인 구분선 · 굵고 좁은 제목.
**DON'T** —
- ❌ 글라스모피즘 / `backdrop-blur`
- ❌ 그라데이션 배경 · 그라데이션 텍스트 · 그라데이션 보더
- ❌ 글로우(빛나는) 그림자 · inner glow
- ❌ 별가루/파티클/장식 애니메이션
- ❌ Inter / DM Sans / 시스템 기본 폰트
- ❌ 경쟁하는 다중 액센트 색
- ❌ 튕기는(bounce) 애니메이션
- ❌ 둥근 카드 남발("card soup")

## 8. 반응형 & 인터랙션
팝업 380 고정, 탭은 560까지 유동. 터치 타깃 ≥36px. 트랜지션 120ms ease, opacity/배경색만(형태 변형 최소).

## 9. 에이전트 프롬프트 가이드
> "[컴포넌트]를 이 DESIGN.md대로 만들어. 플랫 Spotify 네이티브. 그라데이션/블러/글로우 금지. 초록은 주요 액션에만. 앨범아트가 유일한 색."
